CSS Properties Playground

Color

The color property sets the color of the text. You can use predefined color names, hexadecimal values, or RGB values.

Sample Text

Background-color

The background-color property sets the background color of an element. It works with the same color values as the color property.

Font-size

The font-size property sets the size of the text. You can specify the size in pixels, percentages, or other relative units.

Sample Text

Font-weight

The font-weight property sets the boldness of the text. It accepts values from 100 to 900.

Sample Text

Text-align

The text-align property sets the alignment of the text within its container. It can be set to left, right, center, or justify.

Sample Text

Padding

The padding property sets the space between the content of an element and its border.

Sample Text

Margin

The margin property sets the space outside the border of an element.

Sample Text

Border

The border property sets the width, style, and color of an element's border.

Sample Text

Width

The width property sets the width of an element.

Height

The height property sets the height of an element.

Background-image

The background-image property sets one or more background images for an element.

Opacity

The opacity property sets the transparency of an element. It takes a value between 0.0 (completely transparent) and 1.0 (completely opaque).

Sample Text

Text-decoration

The text-decoration property sets the decorations applied to the text. It can be used to add underline, overline, line-through, or none.

Sample Text

Transform

The transform property applies a 2D or 3D transformation to an element. It can be used to rotate, scale, skew, or translate an element.

Sample Text

Box-shadow

The box-shadow property adds a shadow effect to an element. It allows you to set the shadow's color, offset, blur, and spread.
